JavaScript en el Desarrollo Web

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite añadir interactividad a tu sitio web. Es uno de los lenguajes más populares y es compatible con todos los navegadores modernos.

Cómo utilizar JavaScript en una página web

Para utilizar JavaScript en una página web, puedes escribir tu código dentro de la etiqueta <script>. Aquí tienes un ejemplo:

<script>
    console.log('Hola, JavaScript!');
</script>

Manejo de Variables en JavaScript

Las variables en JavaScript se utilizan para almacenar datos que pueden cambiar durante la ejecución del programa. Puedes declarar una variable utilizando las palabras clave var, let, o const. Ejemplo:

let nombre = 'Edwin'; 
const edad = 41;
